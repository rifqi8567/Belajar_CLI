CLI
adalah singkatan dari Command Line Interface

apa itu CLI

CLI (Command Line Interface) adalah antarmuka berbasis teks yang memungkinkan pengguna untuk berinteraksi dengan komputer atau sistem operasi melalui perintah-perintah yang diketikkan.

Shell
adalah program yg menerima perintah kita kemudian meneruskan perintah tersebut agar bisa di eksekusi oleh sistem

Shell adalah program perantara antara pengguna dan sistem operasi, yang:

Menerima perintah dari pengguna (biasanya dalam bentuk teks),

Meneruskannya ke sistem operasi untuk dieksekusi,

Lalu menampilkan hasilnya kembali ke pengguna.

Shell
ada yg berbentuk text dan ada yg bentuk nya grafik
